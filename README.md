# 🔐 Login Form UI (HTML & CSS)

A modern and responsive Login Form UI built using pure HTML5 and CSS3. This project demonstrates a clean design with smooth input animations and a gradient background.

## 📌 Features
- 🎨 Beautiful gradient background
- 📱 Responsive centered layout
- ✨ Animated input fields
- 🔑 Username & Password form
- 🔗 Forgot Password option
- 📝 Signup link section
- ⚡ Pure HTML & CSS (No JavaScript)

## 🚀 Technologies Used
- HTML5
- CSS3

## 🛠️ How to Run
- live link https://janendrasingh.github.io/Login-Format/

## ✨ Customization
- Edit style.css to change colors and layout
- Modify index.html to add validation or backend integration
